# 2013年7月15日于七渡孤山寨

```
芳雨纷菲晨日拱，小别馨室到城东。
无边清秀好风景，幽幽水溪坠崖中。
浮萍映碧峰高耸，近山沁怡云远空。
天丽浑然最痴迷，巧工精绝蹑仙踪。
```
